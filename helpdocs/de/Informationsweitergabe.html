Informationsweitergabe

Einige wenige Inhalte des Fahrtenbuchs sind so wenig kritisch, dass sie öffentlich zugänglich gemacht werden können und eingebunden werden können auf anderen Web-Sites, etwa auf dem Standard Webauftritt des Vereins.

Was so ohne Autorisierung einsehbar ist, kann hier festgelegt werden. Die Information kann mit einem iframe, dessen URL auf deineEfaCloudDomain.org/public/info.php zeigt, eingebunden werden. Ein Beispiel:

<iframe src="../public/info.php?type=onthewater&mode=7" title="Boote auf dem Wasser" style="width: 100%; border: none"></iframe>
