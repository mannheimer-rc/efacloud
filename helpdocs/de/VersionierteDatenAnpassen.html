Ändern von versionierten Daten

Efa verwendet Tabellen mit Einträgen, die mehrere Versionen mit aufeinanderfolgender Gültigkeit haben. Die Tabellen sind: Gruppen, Gewässer, Personen und Ziele.

In diesen Tabellen kann grunsätzlich nur die letzte Version geändert werden. Es gibt drei Änderungsmodi:

  1. Inhalte ändern:
    Der Gültigkeitszeitraum bleibt gleich, Der Datensatz wird inhaltlich geändert (z. B. neue e-Mail Adresse.)
  2. Gültig Bis ändern:
    Das Ende des Gültigkeitszeitraums wird verändert. Es werden keine Inhalte geändert, ggf. eingetragene Änderungen werden ignoriert. Um unbeschränkte Gültigkeit zu erreichen, muss das Datum der Gültigkeit leer sein.
  3. eine neue Version erzeugen: der 'Gültig Bis' Wert des Datensatzes wird so gesetzt wie angegeben, es wird eine neuen Version erzeugt die direkt im Anschluss und dann unbegrenzt gültig ist als Kopie der aktuellen Version. Es werden keine Inhalte geändert, ggf. eingetragene Änderungen werden ignoriert.
