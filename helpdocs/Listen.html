Anzeige von Listen

Tabellen werden in efaCloud als Listen zugänglich gemacht, die jeweils gefiltert sind. Eine Liste bietet folgende Funktionen an:

Sortieren

Die oberste Zeile enthält die „Spaltenköpfe“. Mit einem Klick au den Spaltenkopf wird danach aufsteigend sortiert. Mit jedem weiteren Klick wird die Sortierreihenfolge umgekehrt.

Filtern

Unterhalb der Liste gibt es die Möglichkeit, die Datensätze zusätzlich nach einem Dtenfeld zu filtern auf Datensätze, die in dem Datenfeld dem angegebenen Wert entsprechen. So filtert in der Liste der Fahrten der Filter auf „LogbookName“ mit dem Wert „201*“ auf alle Datensätze der 2010er Jahre

Herunterladen

Die aktuell gefiterte Liste kann über den Link ium Text unter der Liste heruntergeladen werden. Das Format ist eine csv-Datei, die in einem Zip-Archiv komprimiert ausgeliefert wird.

Einzelne Datensätze anzeigen

In manchen Listen gibt es eine Spalte, über die man zur Anzeige des Datensatzes gelangt. Wählt man zum Beispiel im Menü „Sportbetrieb“ die Schadensmeldungen, kann mit Klick auf einen Eintrag n der Spalte „Damage“ die Schadensmeldung angezeigt werden.
